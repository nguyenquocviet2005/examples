# AnythingLLM - Quick Start

## Setup

```bash
yarn setup
```

## Run (Development)

```bash
# Run all services at once
yarn dev:all

# Or run separately in 3 terminals:
yarn dev:server     # Terminal 1
yarn dev:collector  # Terminal 2
yarn dev:frontend   # Terminal 3
```

## Run (Production)

```bash
yarn prod:server
yarn prod:frontend
```
