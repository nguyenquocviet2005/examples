# Claude Skills

Demo scripts for presentations and spreadsheets using Claude.

## Setup

```bash
npm install
# or
python -m venv .venv && source .venv/bin/activate
pip install openpyxl
```

## Run

```bash
# Create presentation
node create_presentation.js

# Create spreadsheet
python create_sample_spreadsheet.py

# Run demo
node demo_presentation.js
```
