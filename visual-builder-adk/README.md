# Visual Builder ADK

Visually built agents using Google ADK.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install google-adk
```

## Configure

Create `.env` file with your API keys.

## Run

```bash
source .venv/bin/activate

# Run an agent
adk run my_agent/
# or
adk run new_agent/
# or
adk run visually_built_agent/
```
