# Scrapling Test

Web scraping with Scrapling library (stealth mode, Cloudflare bypass).

## Setup

```bash
python -m venv .venv
source .venv/bin/activate
pip install scrapling playwright
playwright install
```

## Run

```bash
source .venv/bin/activate
python test.py
```

## Jupyter Notebook

```bash
jupyter notebook Scrapling_Kernel_Troubleshooting.ipynb
```
